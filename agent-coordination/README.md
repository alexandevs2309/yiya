# Agent Coordination — D' Yiya Restaurants

This directory tracks project state for opencode subagents.

- `STATE.md` — Current project state and phase
- `TASK_BOARD.md` — In progress / backlog / blocked items
- `ROLES.md` — Agent role descriptions

When a subagent finishes a task, update TASK_BOARD.md to move the item.
When phase changes, update STATE.md with new phase number and status.
